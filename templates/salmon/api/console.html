Console Reporting

Salmon provides some helpful functions for reporting to the console. Globally a log() function provides safe logging to the console. In addition throughout the framework we have helpful colour coded error messages for logging errors and likely solutions.

log( ...msg ) Returns: undefined
Ensures a console exists before calling console.log
...msg : Any Type
Comma separated arguments of any type.
fin.log.error( msg ) Returns: undefined
Ensures a console exists before calling console.log with a colour coded error string.
msg : String
The message to log.
fin.log.solution( msg ) Returns: undefined
Ensures a console exists before calling console.log with a colour coded solution string.
msg : String
The message to log.
