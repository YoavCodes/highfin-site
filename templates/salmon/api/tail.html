Tail Server

The core tail server is in the /node folder. You run it via the salmon.js file in the root folder.

The tail server provides the propellent for your Salmon app. It'll load up and cache any front-end templates, provide a flexible api and routing structure, control-flow, an in-process persistent data store that's highly compatable with the fin data model, a great balance between flexibility and automation in structuring your back-end code, and more.
