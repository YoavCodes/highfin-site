Introduction

Aquatic is a multi-server continous deploy PaaS hosting infrastructure.

It includes:

  • Octopus - Git server and brains of the infrastructure
  • Squid - Intelligent reverse proxy
  • Sharks - Docker container deploy boxes
  • Jellyfish - Mini reverse proxy that lives inside your container and allows for complex routing between instances

With Aquatic you can easily deploy multiple apps and instances of just about anything you like and wire it all up together using the same -.json file you use to specify your dev environment.
