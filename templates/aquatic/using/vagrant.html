Vagrant

The HighFin GitHub repository has a Vagrantfile configured to setup and run multiple VMs locally.

Assuming you have Vagrant and Virtualbox installed on your system, just clone HighFin, cd into the folder and run the following commands.

vagrant up octopus - will create a new VM and configure it to run octopus

vagrant ssh octopus - will ssh into your octopus VM

guppy run octopus - will actually run the octopus server inside the VM re-install/re-run octopus on changes to the octopus source code.

repeat those steps in new terminal windows for squid, shark0, shark1, and guppy.

The guppy task runner is currently required to create new projects on octopus, it also handles pushing for continuous deploys and at dev time. If you already have a project in a fry-box you can use that guppy to configure your project to push to your local octopus instead of spinning up the guppy vagrant VM suggested above.
