Vagrant

The HighFin GitHub repository has a Vagrantfile configured to setup and run multiple VMs locally.

Assuming you have Vagrant and Virtualbox installed on your system, just clone HighFin, cd into the folder and run the following commands.

vagrant up guppy - will create a new VM and configure it to run guppy

vagrant ssh guppy - will ssh into your guppy VM

guppy run guppy - will actually install and run guppy, re-installing on file change.
