Vagrant

The HighFin GitHub repository has a Vagrantfile configured to setup and run multiple VMs locally.

Assuming you have Vagrant and Virtualbox installed on your system, just clone HighFin, cd into the folder and run the following commands.

vagrant up fishtank - will create a new VM and configure it to run fishtank

vagrant ssh fishtank - will ssh into your fishtank VM

guppy run fishtank - will actually run the fishtank server inside the VM and re-install/re-run fishtank on changes to the fishtank source code.
